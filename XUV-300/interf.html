Specifications


Engine

Type 1

  • Type : TCMPFI engine / mStallion -TGDi engine
  • Capacity : 1.2 L
  • Max. Power : 81kW @5000 r/min | 96 kW @ 5000 r/min
  • Max. Torque : 200 Nm @ 1500-3500 r/min | 230 Nm @ 1500-3750 r/min
  • Transmission Type : 6-speed Manual/ 6-speed AutoSHIFT

Type 2

  • Type : Turbo Diesel with CRDi
  • Capacity : 1.5 L
  • Max. Power : 85.8kW @3750 r/min
  • Max. Torque : 300 Nm @ 1500-2500 r/min
  • Transmission Type : 6-speed Manual/ 6-speed AutoSHIFT

Dimensions

  • LxWxH : 3995 mm x 1821 mm x 1627 mm
  • Wheelbase : 2600 mm

Suspensions

  • Front Suspension : MacPherson Struts Anit-roll Bar
  • Rear Suspension : Twist Beam Suspension with Coil Spring

Brake

  • Front & Rear Disc Brakes

Wheels

  • 205/65 R16 (Alloys)- For W8 & W8 (Optional Pack)
  • 205/65 R16 (Steel)- For W4 & W6

Fuel Tank Capacity

  • 42 Litre
